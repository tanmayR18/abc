
# Dicord Clone

## Tech Stack

**Client:** Html and  Tailwind CSS

**Server:** None

## Usage

Click on the link given below or follow the instrucations given below to run the project locally.




## Run Locally

Clone the project

```bash
  npm install
```

Go to the project directory

```bash
  npm run start
```
